Sobre o Projeto

O Aumigo Aqui é uma plataforma fictícia desenvolvida como trabalho final para a disciplina de Desenvolvimento Web por Pedro Augusto Marques Kloster, acadêmico do curso de Engenharia de Computação da Universidade Estadual de Ponta Grossa.

O objetivo principal desta aplicação é simular um ambiente de adoção responsável de cães, conectando protetores de animais e ONGs a pessoas dispostas a dar um novo lar para esses pets.

Projeto desenvolvido por Pedro Augusto Marques Kloster, acadêmico de Engenharia de Computação da Universidade Estadual de Ponta Grossa, para a disciplina Desenvolvimento Web. Os cães registrados neste site não estão para adoção e os contatos cadastrados são fictícios, sendo apenas elementos para ilustração e demonstração da aplicação web.
